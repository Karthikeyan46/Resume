# Portfolio Website Walkthrough - Mr. Karthikeyan S

I have successfully created a premium, professional portfolio website for Mr. Karthikeyan S. The website is designed to highlight his 9+ years of academic experience, research publications, and technical expertise.

## Key Features

- **Premium Design**: A modern dark-themed aesthetic with a slate/indigo palette, glassmorphism, and smooth scroll-triggered animations.
- **Academic Focus**: Dedicated sections for Experience, Publication (including his CRC Press book), and Research profiles (ORCID, Google Scholar).
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Elements**: Sticky navigation, scroll-reveal effects, and smooth anchor link transitions.

## Implementation Details

- **Structure**: Semantic HTML5 for better SEO and accessibility.
- **Styling**: Vanilla CSS using custom properties for easy theme management.
- **Logic**: Vanilla JavaScript for scroll animations and interactive navigation.
